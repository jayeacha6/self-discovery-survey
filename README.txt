Framer-compatible static survey package

Files:
- survey.html
- mental-overload.html
- emotional-fatigue.html
- nervous-system-overactivation.html
- stress-pressure.html
- inner-disconnection.html
- restlessness-tension.html

How it works:
1. Upload all HTML files to the same folder/location.
2. Link your Framer button to survey.html.
3. User completes the survey.
4. JavaScript saves Q2/Q3/Q4 personalization in localStorage.
5. Q1 redirects the user to the matching reflection landing page.

Recommended Framer use:
- Host the files externally and embed survey.html using an iframe.
- Or place survey.html and the result pages on your custom hosting and link from Framer.
